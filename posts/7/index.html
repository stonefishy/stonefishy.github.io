Backslash Escapes in Markdown

Oct 28th, 2014 12:00 am

Markdown allows you to use backslash to generate literal characters that possesses special meaning in Markdown syntax.

Markdown provides backslash escapes for the following characters:

  • \ blackslash
  • ` backtick
  • * asterisk
  • _ underscore
  • {} curly braces
  • [] square brackets
  • () parentheses
  • # hash mark
  • + push sign
  • - minus sign (hyphen)
  • . dot
  • ! exclamation mark
